Optimization Problems

Finding maximums and minimums in real-world scenarios.

üí° Key Concepts

The Protocol:
  1. Draw a picture.
  2. Constraint: What is limited?
  3. Objective: What do you maximize/minimize?
  4. Solve constraint for one variable.
  5. Plug into Objective function.
  6. Find derivative, set to 0.
  7. Verify: Check critical numbers & endpoints.
Derivative Tests:
  • $f'(x)$ changes + to - $\to$ MAX
  • $f'(x)$ changes - to + $\to$ MIN
  • $f''(c) > 0$ (Concave Up) $\to$ MIN
  • $f''(c) < 0$ (Concave Down) $\to$ MAX

Practice Problems

Q1 (2 pts): Second Derivative Test Concept Show/Hide Answer ‚ñº

True or False: If c is a critical number for $f(x)$ and $f''(c)=0,$ then $f(x)$ has neither a local maximum nor a local minimum at (c, f(c)).

Answer: (B) False

If $f''(c)=0$, the Second Derivative Test fails (is inconclusive). It does not mean there is no max/min. You must go back and use the First Derivative Test.
Example: $f(x)=x^4$ at $x=0$. $f''(0)=0$, but it is a minimum.

Q7 (4 pts): Graph Analysis Show/Hide Answer ‚ñº

The graph of the derivative $y=f'(x)$ is shown. At which value of x does f have a local maximum?

Answer: (D) $x=4$

A local maximum occurs when $f(x)$ goes from increasing to decreasing. This means $f'(x)$ must change from Positive to Negative. Look at the graph: the line crosses the x-axis from above to below exactly at $x=4$.

Q24 (10 pts): "Captain Efficiency" Show/Hide Solution ‚ñº

Scenario: Minimize energy. Swimming costs 2x as much as running. Canal width $\sqrt{300}$. Run $x$, then swim to point 40m downstream.

1. Variables:
  • Run Distance: $x$
  • Remaining distance on bank: $40-x$
  • Swim Distance (Hypotenuse): $\sqrt{(\sqrt{300})^2 + (40-x)^2} = \sqrt{300 + (40-x)^2}$
2. Objective Function:

$E(x) = (1 \cdot x) + (2 \cdot \text{Swim Distance})$

$E(x) = x + 2\sqrt{300 + (40-x)^2}$

3. Derivative & Solve:

$E'(x) = 1 + 2 \cdot \frac{1}{2\sqrt{...}} \cdot 2(40-x)(-1) = 0$

Simplify: $1 = \frac{2(40-x)}{\sqrt{300+(40-x)^2}}$

Square both sides: $300 + (40-x)^2 = 4(40-x)^2$

$300 = 3(40-x)^2 \implies 100 = (40-x)^2$

$10 = 40-x \implies x = 30$

Final Answer: Run 30 meters.

FTC & Integrals

Connecting derivatives and integrals.

üìú The Theorem

Part 1: Derivative of Integral

$\frac{d}{dx} \int_a^x f(t) dt = f(x)$

It just "cancels" the integral symbol.

With Chain Rule:

$\frac{d}{dx} \int_a^{g(x)} f(t) dt = f(g(x)) \cdot g'(x)$

Part 2: Evaluation

$\int_a^b f(x) dx = F(b) - F(a)$

Find antiderivative F, plug in top, subtract bottom.

Q11 (4 pts): FTC Part 1 Application Show/Hide Answer ‚ñº

Consider $g(x)=\int_{4}^{x}\frac{1}{1+t}dt$. What is $g'(2)$?

Answer: (C) 1/3

Using FTC Part 1, the derivative $g'(x)$ is simply the integrand evaluated at $x$.

$g'(x) = \frac{1}{1+x}$

$g'(2) = \frac{1}{1+2} = \frac{1}{3}$

Q12 (4 pts): Visualizing Integrals Show/Hide Answer ‚ñº

Evaluate $\int_{1}^{8}f(x)dx$ using the graph below.

Answer: (C) -2

Calculate Area between curve and x-axis.
Above axis = Positive. Below axis = Negative.

  • Area 1-3 (Above): Triangle + Trapezoid $\approx +2.5$
  • Area 3-7 (Below): Large region $\approx -5$
  • Area 7-8 (Above): Small triangle $\approx +0.5$
Q15 (4 pts): Definite Integral Show/Hide Answer ‚ñº

$\int_{-1}^{2}x^{3}dx$

Answer: (C) 15/4

Antiderivative of $x^3$ is $\frac{x^4}{4}$.

Evaluate: $[\frac{2^4}{4}] - [\frac{(-1)^4}{4}] = \frac{16}{4} - \frac{1}{4} = \frac{15}{4}$.

Substitution Rule

Reverse Chain Rule & changing variables.

üîÑ U-Sub Steps

  1. Pick $u$: Usually inside a root, parenthesis, or exponent.
  2. Differentiate: Find $du$. Solve for $dx$.
  3. CRITICAL: Change Bounds!
    For $\int_a^b$, calculate $u(a)$ and $u(b)$.
  4. Substitute: Replace all x's with u's.
  5. Integrate: Use power rule.
Q22 (4 pts): Changing Bounds Show/Hide Answer ‚ñº

Integral $\int_{2}^{3}\frac{x}{6+x^{2}}dx$ with sub $u=6+x^{2}$. Equivalent integral?

Answer: (E) $\int_{10}^{15}\frac{1}{2u}du$

1. Find du: $du = 2x dx \implies \frac{1}{2}du = x dx$.

2. Change Bounds:

  • Lower: $x=2 \to u = 6+(2)^2 = 10$
  • Upper: $x=3 \to u = 6+(3)^2 = 15$

3. Result: $\int_{10}^{15} \frac{1}{u} (\frac{1}{2}du)$

Q23 (10 pts): Tricky Indefinite Sub Show/Hide Solution ‚ñº

Evaluate $\int x\sqrt{1-3x}dx$

1. Let $u = 1-3x$. Then $du = -3dx \implies dx = -\frac{1}{3}du$.

2. Solve for x: We have an extra 'x' outside.
$u = 1-3x \implies 3x = 1-u \implies x = \frac{1-u}{3}$.

3. Substitute: $\int (\frac{1-u}{3}) \sqrt{u} (-\frac{1}{3}du)$

4. Clean up constants: $-\frac{1}{9} \int (1-u)u^{1/2} du$

5. Distribute & Power Rule: $-\frac{1}{9} \int (u^{1/2} - u^{3/2}) du$

Don't forget +C at the end!

Riemann Sums

Approximating area with rectangles.

üìè Formulas

Width ($\Delta x$):

$\frac{b-a}{n}$

Right Endpoint ($x_i$):

$a + i\Delta x$

Limit Definition:

$\lim_{n \to \infty} \sum_{i=1}^n f(x_i) \Delta x$

Q9 (4 pts): Limit Definition Show/Hide Answer ‚ñº

Which limit represents $\int_{2}^{8}\tan~x~dx$?

Answer: (D)

1. Width $\Delta x = \frac{8-2}{n} = \frac{6}{n}$.
2. Endpoint $x_i = 2 + \frac{6i}{n}$.
3. Plug into sum: $\sum \tan(2 + \frac{6i}{n}) \cdot \frac{6}{n}$

Q10 (4 pts): Calculate R4 Show/Hide Answer ‚ñº

Find Right Hand Estimate $R_4$ for $[3, 15]$.

Answer: (D) 33

1. Width: $(15-3)/4 = 3$.
2. Evaluate at Right Endpoints: $x=6, 9, 12, 15$.
3. Heights: $4, 3, 1, 3$.
4. Area: $3 \times (4+3+1+3) = 3 \times 11 = 33$.

Antiderivatives

Working backwards from the derivative.

‚Ü©Ô∏è Rules

Power Rule:

$\int x^n dx = \frac{x^{n+1}}{n+1} + C$

Trig:

$\int \sin x = -\cos x$

$\int \cos x = \sin x$

Don't forget +C!
Q14 (4 pts): Finding C Show/Hide Answer ‚ñº

Given $f'(x)=3+2\sin x$ and $f(0)=7$. Find $f(\pi)$.

Answer: (B) $3\pi + 11$

  1. Integrate: $f(x) = 3x - 2\cos x + C$
  2. Plug in (0,7): $3(0) - 2(1) + C = 7 \to -2 + C = 7 \to C=9$
  3. Final Function: $f(x) = 3x - 2\cos x + 9$
  4. Plug in $\pi$: $3\pi - 2\cos(\pi) + 9 = 3\pi - 2(-1) + 9 = 3\pi + 11$
